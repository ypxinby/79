# 云台调试记录

## 2026-07-16 P1 单轴冒烟测试

### 问题现象

PITCH 轴接线后，云台步进电机最初没有反应。

### 最终确认的正确接线

```text
MSPM0 拓展板 PB4  -> D36A ST1
MSPM0 拓展板 PA1  -> D36A DIR1
MSPM0 拓展板 PA2  -> D36A EN1
MSPM0 拓展板 GND  -> D36A GND
MSPM0 拓展板 5V   -> D36A 5V
```

### 根因

1. 原方案误用了 `PA3` 作为 PITCH STEP，但当前天猛星 2026 小车拓展板没有引出 `PA3`。
2. `PA3` 在 MSPM0G3507 上属于 LFXIN 低频晶振相关引脚，应避开，不作为云台 GPIO/PWM 使用。
3. D36A 驱动板 `EN1` 为高电平工作、低电平休眠，而测试代码最初按低有效使能处理，导致驱动没有被使能。

### 修正

1. PITCH STEP 从 `PA3` 改到 `PB4`，接 D36A `ST1`。
2. PITCH DIR 保持 `PA1`，接 D36A `DIR1`。
3. PITCH EN 保持 `PA2`，接 D36A `EN1`。
4. 将测试代码中的使能极性改为高有效：

```c
#define GIMBAL_EN_ACTIVE_LOW (0)
```

### 验证结果

重新编译下载后，PITCH 步进电机已经可以转动。说明当前 P1 的基础 STEP/DIR/EN 链路可用。

## 2026-07-16 P3 反向步数验证

### 验证结果

在当前 16 细分设置下，`-800 step` 的反向动作已经实测成立，电机大约反向转动 90° 后自动停止。

### 结论

1. 正向和反向的 `STEP/DIR/EN` 链路都正常。
2. 固定步数停止逻辑在正反方向上都可用。
3. 当前底层适合进入“步数 -> 角度”阶段。

## 2026-07-16 P5 OLED 按键小角度验证

### 修改内容

1. 新增公开云台接口层 `gimbal.h/.c`，业务代码通过 `Gimbal_MoveRelativeDeg()` 发起相对角度运动。
2. 关闭上电自动运行：`FEATURE_GIMBAL_TEST_AUTO_RUN = 0`。
3. 新增 OLED `GIMBAL` 页面，用于手动验证小角度运动。

### 按键定义

```text
K1 短按：STATUS -> HEADING -> OBSTACLE -> GIMBAL -> STATUS 页面循环
K2 短按：PITCH +5°
K3 短按：PITCH -5°
K2 长按：PITCH +30°
K3 长按：PITCH -30°
```

### OLED 显示

```text
T10: 目标相对角度 x10
C10: 当前已完成相对角度 x10
RUN: 是否正在运动
D:   当前方向
OK:  是否到达目标
```

## 2026-07-16 P6 5ms 云台控制入口

### 修改内容

1. 保留 `Gimbal_Tick100us()` 作为 STEP 细脉冲入口。
2. 新增 `Gimbal_Update5ms()` 作为后续实时追踪控制入口。
3. `empty.c` 中由 100us SysTick 派生 5ms pending 计数，主循环先处理云台 5ms，再处理原有 20ms 小车任务。
4. 5ms pending 上限为 2，避免 OLED 或其他 20ms 任务阻塞后补太多历史控制 tick。

### 当前验证方式

OLED `GIMBAL` 页面新增 `T5` 显示。进入页面后，`T5` 应持续增长，表示 5ms 云台控制入口正在被主循环服务。

### 当前边界

`Gimbal_Update5ms()` 目前只作为控制心跳和后续闭环入口，不执行 OLED、UART、软件 I2C，也不改变现有小车任务链顺序。

## 2026-07-16 P7 Stop/Hold/Release 状态收拢

### 修改内容

1. 公开云台接口新增：

```text
Gimbal_StopHold()
Gimbal_Release()
```

2. 云台反馈新增三态模式：

```text
REL: Released，驱动释放，EN 关闭
HLD: Holding，驱动使能但不运动
MOV: Moving，正在执行相对角度运动
```

3. OLED `GIMBAL` 页面第 4 行显示：

```text
M: 模式
E: EN 使能状态
T5: 5ms 控制心跳
```

### 按键定义

```text
K1 短按：切换到下一个页面
K1 长按：进入参数菜单，保持原工程操作习惯
K2 短按：PITCH +5°
K3 短按：Release，释放电机
K2 长按：StopHold，停止并保持
K3 长按：Release，释放电机
```

### 验收现象

```text
上电默认 REL，电机可被手动轻轻转动
K2 长按后变 HLD，电机有保持力
K2 短按发运动命令后变 MOV，运动结束自动回 HLD
运动中 K2 长按应立即停止并保持
HLD 状态下 K3 短按或长按应变 REL，电机释放
```

### 后续注意

1. 后续双轴引脚分配必须以天猛星 2026 小车拓展板实际引出引脚为准，不只看 MSPM0G3507 芯片封装能力。
2. 避开 `PA3/PA4` 这类低频晶振相关引脚。
3. D36A 控制逻辑按当前实测记录处理：`ST` 上升沿有效，`DIR` 高低电平切换方向，`EN` 高电平工作。
