轮速和里程公式：

```text
distance_per_pulse_cm = pi * wheel_diameter_cm / encoder_ppr_x2
raw_speed_cmps = signed_delta_pulse * distance_per_pulse_cm * 1000 / elapsed_ms
filtered_speed = previous + 0.35 * (raw_speed - previous)
total_distance = signed_total_pulse * distance_per_pulse_cm
center_distance = (left_total_distance + right_total_distance) / 2
```

映射和控制公式：

```text
target_cmps = normalized_command / 1000 * 60.0
error = ramped_target_cmps - measured_filtered_cmps
feedforward = ramped_target_cmps / 60.0 * 1000 * feedforward_gain
P = kp_used * error
integral += error * dt_s
I = Ki * integral
output = clamp(feedforward + P + I, target_sign_allowed_range)
```

正目标的输出范围被限制为 `0..1000`，负目标限制为 `-1000..0`，因此超速纠偏只会把同方向PWM压到0，不会自动反转制动。

## 12. 当前文档使用原则

- P4、P5、P6.1参数已经形成实车基线，后续任务不得顺手重调。
- 旧`TEST-YH`和组合导航诊断已删除；当前先分别验收`TEST-TURN`和`TEST-HEAD`，不要混合测试。
- 当前没有距离闭环动作；`center_distance_cm`只是观测量，不是动作结束条件。
- OLED当前是只读诊断，但实现上直接读取多个模块Runtime；它不会重新读取传感器，也不会调用`Encoder_GetAndClearPulseDeltas()`。
- `Debug/`为CCS生成目录；本轮文档审计只读取，不手工修改。
